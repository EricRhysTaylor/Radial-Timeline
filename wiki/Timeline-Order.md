# Timeline order

`Timeline order` opens the timeline order normalizer, also called the Timeline Repair wizard.

Use it when your chronology needs a structured cleanup pass instead of manual scene-by-scene edits.

## What It Does

The wizard helps you normalize `When` values in manuscript order, then review the proposed timeline before writing changes back to frontmatter.

It supports:

*   scaffold-based chronology setup
*   anchor date and time selection
*   time-bucket adjustments such as morning, afternoon, evening, and night
*   ripple mode for cascading changes
*   needs-review filtering
*   undo and redo before applying changes

## Workflow

1. Open `Timeline order`.
2. Choose a scaffold pattern and anchor time.
3. Review the proposed scene order and timing.
4. Adjust scenes, buckets, or ripple behavior as needed.
5. Apply the changes back to your scene frontmatter.

This command is for repairing and normalizing chronology, not for auditing contradictions.
