# Scene pulse analysis (subplot order)

`Scene pulse analysis (subplot order)` opens the subplot pulse picker first, then runs pulse analysis for a selected subplot.

Use it when you want to evaluate one subplot arc as its own sequence instead of processing the manuscript as a whole.

## How It Works

The command opens a subplot picker that shows:

*   flagged scenes
*   processable scenes
*   total scenes

From there you can choose:

*   **Process flagged scenes**
*   **Process entire subplot**
*   **Purge all pulse** for that subplot

## Notes

This command is especially useful when a subplot has changed heavily and you want pulse feedback just for that arc.
