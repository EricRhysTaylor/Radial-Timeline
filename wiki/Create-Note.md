# Create note…

`Create note…` opens the guided RT note picker.

Use it when you want to create the right note type without remembering YAML, filenames, or folder conventions by hand.

## What It Creates

The picker is organized into three families:

*   **Scene**: Core scene, scene with advanced properties, screenplay scene, and podcast scene
*   **Manuscript matter**: Front matter, back matter, and `BookMeta`
*   **Story world**: Beat and Backdrop

## What Happens Next

After you choose a subtype, Radial Timeline creates the file in the active book/source folder and opens it immediately.

Scene creation includes built-in scaffolds:

*   **Core scene**: minimal scene properties for straightforward drafting
*   **Scene with advanced properties**: richer scene metadata
*   **Screenplay scene**: screenplay body scaffold plus runtime defaults
*   **Podcast scene**: host/guest script scaffold plus runtime defaults
