# Gossamer analysis

`Gossamer analysis` runs the built-in AI scoring workflow for the active Gossamer signal.

Use it when you want AI-generated beat scoring for the currently selected signal rather than manual entry.

## What It Evaluates

Gossamer analysis works against the active beat system and the active signal:

*   Momentum
*   Tension
*   Activity
*   Interiority

The AI run intentionally does **not** receive ideal-range guidance in its scoring payload, so the result is a fresh read rather than a range-constrained one.
