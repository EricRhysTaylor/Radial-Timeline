# Scene pulse analysis (manuscript order)

`Scene pulse analysis (manuscript order)` opens the pulse processing modal for manuscript-order analysis.

Use it when you want scene pulse feedback based on the manuscript sequence rather than a single subplot.

## Run Modes

The modal supports several pulse modes:

*   process open scenes
*   process flagged scenes
*   process unprocessed scenes
*   reprocess all scenes

This command is the main batch workflow for updating scene hover pulse analysis across the manuscript.

## Notes

Pulse works on scenes in **Working** or **Complete** status and is intended for scene-by-scene editorial feedback, not long-form summary generation.
