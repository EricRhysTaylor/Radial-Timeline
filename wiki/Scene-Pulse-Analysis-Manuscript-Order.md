# Scene pulse analysis (manuscript order)

`Scene pulse analysis (manuscript order)` opens the pulse command panel for manuscript-order analysis.

## Run Modes

The command panel supports several pulse modes:

*   process open scenes
*   process flagged scenes
*   process unprocessed scenes
*   reprocess all scenes
