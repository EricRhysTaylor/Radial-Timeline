## Radial Timeline 6.0.6

This is a visual polish and APR reliability release.

Note: Pandoc PDF Designer is still under development and is not yet released. Use one of the four built-in PDF templates for now.

### New Features

- Added selectable Working-scene patterns in Settings → Core, with new built-in SVG motifs for customizing how active scenes appear on the timeline.
- Added more detail to the Timeline legend, including number-square state guidance for scene interaction and status reading.
- Added clickable APR output paths so generated reports can be revealed directly in the system file manager.

### Improvements

- Refined APR preview and export rendering so shared PNG/SVG output follows the main Timeline visual language more closely.
- Improved campaign publishing controls, including clearer stage reveal states and default-report view behavior.
- Continued hardening the Pandoc PDF Designer internals, including style-wizard copy, template tier behavior, and PDF layout validation.
- Updated license, notice, bundled-font credits, and v7 migration planning docs.

### Bug Fixes

- Fixed APR campaign export and teaser-stage reveal edge cases, including manual stage reveal behavior.
- Fixed Social settings alignment and border color picker behavior.
- Fixed PDF publishing validation edge cases around bundled templates and layout tier access.
